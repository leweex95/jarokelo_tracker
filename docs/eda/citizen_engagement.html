👥 Citizen Engagement Analysis

What you see: Comparison of reporting behavior between anonymous and registered users across five key dimensions.

  • Resolution Rate: How successfully each group's reports get resolved
  • Avg Images: Documentation quality (photos per report)
  • Description Length: Detail level in problem descriptions
  • Urgency Rate: Frequency of urgent issue reporting
  • Updates Received: Institutional communication frequency

Key Insights:

  • Registered users provide more detailed reports and receive slightly better resolution rates
  • Anonymous users make up 62% of all reports, indicating trust/privacy concerns
  • Both groups show similar documentation patterns (image usage)
  • Registered users receive more updates, suggesting better institutional follow-up
