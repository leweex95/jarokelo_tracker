About This Project

The Járókelő RAG System collects issues from járókelő.hu to explore and analyze urban problems in Budapest. It allows:

  • Browsing issues by district, category, and status.
  • Generating summaries and insights from reports.
  • Using AI for semantic search and pattern detection.
